CARVE

WAVE EDITOR · LISA RECORDS

A wave editor that runs entirely in your browser. Audio files you load are processed only on your device — nothing is ever uploaded to or stored on a server. No install, no account.

Features

Loads WAV / AIFF / FLAC / MP3. Range selection with zero-cross snapping (hold Shift for raw selection), fade in/out (5 curves), normalize, gain (±dB), delete / copy / paste (insert at cursor) / extract selection to a new file, loop playback, play-from-start, loop seam check (auditions the end→start transition), markers (add / remove / clear), bar/beat ruler (set BPM and time signature in the status bar), OGG loop tags (LOOPSTART / LOOPLENGTH read into markers on import, and embedded from markers 1–2 on export), click-to-copy time and sample readouts in the status bar, undo/redo, light/dark theme. Exports WAV (16/24-bit + TPDF dither, 32-bit float), OGG Vorbis (VBR quality) and MP3 (CBR bitrate).
